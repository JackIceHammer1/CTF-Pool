What is a website?

A website is a set of files accessible from the internet. A website is typically hosted by a webserver that contains the DNS address and the files used to host the website. DNS stands for Domain Same System and is used to resolve IP addresses to a domain. An IP address is a set of numbers associated with a router, server, or workstation.
